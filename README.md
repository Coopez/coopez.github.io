# Niklas Erdmann — personal website

Personal academic site (research, publications, talks, teaching, CV) built on
the [academicpages](https://github.com/academicpages/academicpages.github.io)
Jekyll template and hosted on GitHub Pages.

- **Edit content:** Markdown in `_pages/`, `_publications/`, `_talks/`,
  `_teaching/`. See `CLAUDE.md` for conventions.
- **Preview locally:** `.\serve.ps1` → http://localhost:4000 (needs Docker).
- **Publish:** push to `main`; GitHub Pages rebuilds automatically.
